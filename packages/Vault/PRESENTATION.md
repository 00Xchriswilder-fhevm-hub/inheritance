## Slide 5: How It Works

### Simple 3-step process:

1. **Create Vault**
   - Encrypt data with AES-256-GCM
   - Encrypt AES key with FHE (via ZAMA)
   - Store on IPFS + Ethereum blockchain

2. **Grant Access**
   - Authorize heir wallet addresses
   - Set release timestamp
   - FHE Access Control Lists (ACL) manage permissions

3. **Unlock**
   - Owner: Access anytime
   - Heirs: Access after release time
   - Automatic decryption via FHEVM

**No passwords. No trust required. Just your wallet.**
